# Manifesto
In nature everything is connected. Inside, outside and inside to outside separated by membranes, or highly differentiated regions separating realms. The partitioning of realms and connecting of realms is life. Biomimetic forms seek to have this strategy as a backbone or more broadly, a shell. Within this shell is connectedness. A reticulated ecosystem. Through physical and time connections by remembered pasts and events. Life is lived, learned and lauded. Sometimes forgotten to relive anew. There must be sensing, filtering, analyzing, remembering, planning, pruning, and learning.

# Mission Statement
The decision to change the level of actuation in a single muscle is connected to a funnel of remembrance and motivation. A muscle also feels its actuation, usage, condition, pain and preservation. A muscle can be measured, filtered and categorized into reactions and remembered to effect a trend of behaviour. Sense of balance, change in balance, flight plan for balance all form the actuations of muscle seeking to achieve balance motivation.

# Notes
- Each board port of the platform has its own project, differences and abilities
- As abilities are developed, they can be shared and moved to the Robot folder
- Network connectivity is possible via Ethernet, USB, and PPP over serial
- Shared network applications include NTP, Telnet, HTTP
- The HTTP web server provides a way to communicate with the board through a command line and observe robotic elements via a line graph and 3D visual element
